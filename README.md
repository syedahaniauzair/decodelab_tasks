DECODE LAB – Data Analytics Internship Report
Overview

This repository/document contains the Data Analytics Internship Report completed during my internship at Decode Lab. The report demonstrates practical data analysis skills using Microsoft Excel, covering data cleaning, exploratory data analysis (EDA), visualization, and business insight generation from an e-commerce sales dataset.

Objectives
The internship project focused on:
Cleaning and preparing raw business data.
Performing Exploratory Data Analysis (EDA).
Creating Pivot Tables and Pivot Charts.
Identifying trends, patterns, and business insights.
Detecting outliers in sales data.
Presenting findings in a professional report.

The following analyses were performed:

Payment Method Analysis
Referral Source Analysis
Order Status Analysis

Outlier Detection

Key Findings
Product Analysis
Printer received the highest number of orders.
Phone received the lowest number of orders.
Demand remained relatively balanced across all products.

Payment Method Analysis
Online payments were the most preferred.
Gift Card was the least used payment method.
Customer payment preferences were fairly balanced.

Referral Source Analysis
Instagram generated the highest number of customers.
Referral generated the fewest customers.
Customer acquisition was distributed across multiple marketing channels.

Order Status Analysis
Cancelled orders were the highest.
Delivered orders were the lowest.
Order status distribution was relatively balanced.

Sales Trend Analysis
2023 recorded the highest total sales.
Sales decreased in 2024.
2025 recorded the lowest sales.
Overall trend indicates declining annual sales.

Outlier Analysis
Using a Box & Whisker plot, two unusually high-value transactions were identified in the TotalPrice column, representing exceptional purchases compared to the rest of the dataset.
